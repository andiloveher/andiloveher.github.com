关于React的一些小知识

react如何决定什么时候重新渲染组件
  • setState
  • shouldComponentUpdate方法默认返回true。告诉react什么时候不需要重新渲染组件，返回false。
用Class创建一个组件渲染在react中经历了什么

  • ReactDom.render调用组件的构造函数，初始化props/state

  • 然后再次调用构造方法render，让react了解该渲染什么

  • 当插入到dom中时，调用组件的生命周期钩子componentDidMount

  • 通过调用setState通知react更新相应的视图

  • 一旦组件被移除，react调用componentWillUnmount钩子

about class

  • 类组件的render和函数式组件的reutrn都是返回一个纯对象，这个纯对象是对dom节点或者组件实例的描述。如果type是string则表示dom节点，如果type是function或者class，则表示组件实例。

  • react是如何知道我们写的是类组件还是函数function的呢，因为经过代码编译都是统一的function。这就涉及到js原型链
